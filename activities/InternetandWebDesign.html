1. Internet -

The internet connects computers so they can talk to each other. It uses special cables to reach faraway places.

2. Web -

Is about the message that we are sharing through the internet. A collection of connected pages that we can visit online.

3. How are the Internet and Web different?

It is different in a way that internet is what's making all the computers around the world connected so that they can share information. While the web is one of the way we use that connection when we open a search engine.

4. ASCII and Unicode -

ASCII composed of 0-127 bits which supports the traditional keyboard keys. While Unicode offers a wider range of characters since it includes Emojis, symbols, and other languages letters.

5. What is UTF -8 -

Widely used format to gather and visualize letters into the computer.

6. URL -

When we expound the term, it is the Uniform Resource Locator. It is a unique identifier of the specific resource in the internet.

7. Domain Name -

Are human-readable, easy to remeber text addresses that map to a web's address. It allows us to access websites.

8. HTTP -

It is a set of rules that control how two computers are supposed to talk to one another.

9. HTTPS -

Just like the HTTP, it is also how they control the communications of computers but HTTPS is an extension of that. It makes sure that our communication in the server is encrypted.

10. IP Address -

In our networking class, it means the Internet Protocol address. It is a unique number assigned to a device in a computer netwrok.

11. DNS -

In our networking class, it means Domain Name System. Instead of typing the IP addresses of every website we visit, DNS lets us type the website name instead, like byuh.com

12. W3C Recommendations -

It is a rule created by World Wide Web Consortium. It explains the best ways to build and use web technologies so that websites will work well.

13. XML, XHTML, HTML, HTML5 -

XML - allows us to set up and save data in a way that can be shared. XHTML - almost the same as HTML but sticter. It is a guide for making web pages that browsers can read easily. HTML - markup language for web that shows how web pages are built. HTML5 - the 5th version of HTML, it is another language to build web pages.
