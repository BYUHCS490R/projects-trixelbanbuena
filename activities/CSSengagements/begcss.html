Engagement Questions

1. Explain the difference between inline, embedded, and external CSS.

  1. Inline CSS → written inside a tag, like p style="color:red;". It only affects that tag.
  2. Embedded CSS → written inside the style tag in the HTML file. It styles just that page.
  3. External CSS → written in a separate css file and linked with ;link. It can style many pages at once.

2. Explain why you should only use external CSS.

  1. It keeps code neat and organized.
  2. One CSS file can control the style for the whole site.
  3. It makes all pages look uniform which is important for organizations.
  4. If you change something, you edit once and it updates everywhere.

3. What is a selector used for?

  1. A selector tells CSS what part of the HTML to style.
  2. Example: p { color: blue; }→ the selector p makes all <p> text blue.
