CSS Selectors

1. What is the difference between an ID and a Class?

Answer:

You can select items in CSS by their name, by an ID, or by a class. You can also use attributes, choose items based on where they sit in the structure (like inside another element), or use special selectors like hover, first-child, or before/after to style certain parts.

2. What is the difference between an ID and a Class?

Answer:

An ID is unique andshould only be used once on a page, while a class can be used on many items. IDs are more powerful in CSS, but classes are better for grouping and styling multiple elements the same way.

3. What is the difference between a Span and a Div?

Answer:

A span is used for small bits of text or content within a line, while a div is used for larger blocks of content. Spans are inline, divs are block-level.

5. What are the different ways to select a color?

Answer:

You can use named colors like red or blue, hex codes like #ff0000, or RGB values with numbers. You can also use RGBA or HSLA if you want transparency, and HSL to set colors by hue, saturation, and lightness.
