When do you use a table? When you need to show data that’s best understood in rows and columns (like numbers, comparisons, or schedules)

When do you not use a table? When you’re just arranging text or images for layout or design — use proper formatting or CSS instead.

When do you need to do extra work for table accessibility? When your table has headers, complex structure, or merged cells — you should add things like header tags (), captions, and ARIA labels so screen readers can understand it
