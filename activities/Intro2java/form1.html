What is JavaScript? How did it get its name?It’s a flexible, high-level programming language used mainly to add interactivity to websites. It was renamed “JavaScript” as part of a marketing move to take advantage of Java’s popularity at the time.

What are two ways you can run JavaScript?You can run it in a web browser or on a server using environments like Node.js.

What was the most popular front-end JS library?jQuery used to be the dominant front-end library.

What is the most popular front-end JS library today?React is currently the most widely used front-end library.
