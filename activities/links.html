1. What is the root? What is the relationship between folder and file names and the path?

Answer: The root is the top level of a website. Folders organize files, and file names are the actual docusments. A path shows the exact location of a file starting from the root.

2. What is a default document?

Answer: The web page a server shows automatically if no file name is given.

3. Explain the difference between an absolute link and a relative link. Why would you use relative links?

Answer: Absolute link includes the full web address, the relative link points to a file based on where you move it to a different server.

5. List the 3 rules with relative links.

Answer:

  1. When I want to link a file in the same folder, I will just use the filename.
  2. If the file is in a child folder, I need to include the folder name
  3. If the file is in a parent folder, need to use ../ to go up one level.
